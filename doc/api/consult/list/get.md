#### 地址
`/api/consult/list/get`

#### 描述
获取用户是否咨询列表

#### 参数
|参数名|类型|是否必须|描述|
|---|---|---|---|
|userId|string|T|用户Id|


示例：
```
{
  userId: 'dfdf'
}
```

#### 成功响应数据格式
```
{
  code: 0,
  msg: "ok",
  data: {
    first: {  //第一次提问
```
